Password Generator

You must complete all the security questions to recieve the potential password list. You then must figure out which password is valid based on model below.

A-Z means any capital letter from the alphabet.
0-9 means any digit.
a-z means any lower-case letter from the alphabet.

There are certain rules for the password to be accepted.

In a loop any quantity of letters or digits can be used several times. In this loop you are allowed to use zero, one or more capital letters.

An edge means that the you have to use exactly one letter or digit. This edge demands one lower-case letter.
